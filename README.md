# README #

股票量化分析工具.此仓库中提供了一批股票分析与报告生成相关的工具脚本。

### 工具脚本说明 ###

| 脚本 | 功能 |
| :-------- | --------: |
| stockFillData.js | 分析当日的活跃股票 | 

#### 活跃股票监控
查看当天哪支股票交易量大（成交额/市值），如果当日涨跌幅也大，说明这支股票当天很活跃。

**运行**
node stockFillData.js

**数据**
* data/xlsx/watch_stock.xlsx
